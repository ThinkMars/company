# Free Theme Ball

## Install

```bash
pnpm install free-theme-ball
```

## Usage

### Vue 3

```ts
import { createApp } from 'vue'
import { VueThemeBallPlugin } from 'free-theme-ball'
import App from './App.vue'
const app = createApp(App)
app.use(VueThemeBallPlugin)
app.mount('#app')
```
