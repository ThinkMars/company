# 类chatgpt的UI界面

## 仓库

[want-chat](https://github.com/ThinkMars/company/tree/main/packages/want-chat)

## 介绍

- 基于 [Ant Design](https://ant-design-x.antgroup.com). 一个专为AI chat设计的UI框架。
- 基于 [Zhipu GLM](https://bigmodel.cn). 清华大学出品的世界上最好的大模型之一。
